== Overview

These tutorials are a guided path through the Q library, ordered from the
simplest to the most involved. Each one is a small, self-contained program in
its own directory under `example/`: one target, one source file, and (when it
reads audio) its own `audio/` directory whose location is baked into the
executable at build time. Every tutorial therefore runs from *any* working
directory; nothing depends on where you launch it from.

----
example/<name>/
   CMakeLists.txt    one target: example_<name>
   <name>.cpp
   audio/            only the audio this tutorial reads (when it reads any)
example/common/      shared helpers (device pickers, keyboard input)
----

Build them all (they are ON by default), then run any one directly:

```sh
cmake -B build
cmake --build build
build/example/sin_osc/example_sin_osc
```

== The Path

Work through the tutorials below in order for a guided tour, or jump to the one
you need; each page links to the next at the bottom. The {tutorial-sin_osc} is
the shortest path to your first sound and the {tutorial-delay} your first effect,
each in a handful of lines, proof of how little code Q needs to make sound.
