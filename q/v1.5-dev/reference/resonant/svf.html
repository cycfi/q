State Variable Filter

Overview

The svf is a topology-preserving-transform (TPT) state-variable filter: the continuous-time analog SVF discretized with trapezoidal integration, with the unit-delay-free feedback resolved analytically (zero-delay feedback). It is based on Andrew Simper’s Solving the continuous SVF equations using trapezoidal integration (© Andrew Simper, Cytomic, 2013) and Vadim Zavalishin’s The Art of VA Filter Design.

The filter is two integrators in a loop with a single damping feedback. Each integrator is a building block with gain g; the input summer subtracts the lowpass and the damped (× k) bandpass. The three primary outputs are taps at the summer (highpass), after the first integrator (bandpass), and after the second (lowpass); the notch, peak, and allpass are linear combinations of those taps and the input.

Figure 1. Building blocks and output taps: two integrators, a damping feedback, and the HP/BP/LP taps (notch/peak/allpass are sums of these).

A single tick produces every second-order response from the same two states: lowpass, bandpass, highpass, notch, peak, and allpass.

Figure 2. Left: the responses available from one filter. Right: the lowpass at rising Q, whose peak height equals Q.

The parametrization is exact. The cutoff coefficient is the bilinear prewarp g = tan(pi * fc / fs), so the actual cutoff (the -3 dB / resonant point) lands precisely at fc at any frequency up to Nyquist. Resonance is the true quality factor Q (internally k = 1/Q): Q = 1/sqrt(2) is maximally flat (Butterworth), larger Q sharpens the peak, and as Q → inf the filter self oscillates. A normalized 0..1 resonance control is also provided for synth-style sweeps.

For modal synthesis the filter may also be placed by center frequency and amplitude decay time constant tau (the 1/e time of the impulse-response envelope), the two numbers a mode table specifies. The mapping follows from the resonator bandwidth relation BW = 1/(pi*tau): Q = pi * f * tau (the free function q_from_decay). Excited by an impulse, the filter rings at f with an envelope that decays by 1/e every tau seconds; a bank of these is the classic modal voice. The bandpass and lowpass taps are 90 degrees apart at resonance, so bg * bandpass + lg * lowpass realizes an arbitrary complex per-mode transfer gain, magnitude and phase, with two multiply-adds on taps one tick already computes.

Because the two states are integrator outputs with a consistent physical meaning, the cutoff may be changed every sample and the filter stays stable and free of zipper noise, even at high resonance and into self-oscillation. This is the property a direct-form Biquad lacks, and the reason the svf is the right choice for an envelope- or LFO-swept filter. The filter is unconditionally stable for g > 0 and k >= 0.

Cost is about six multiplies per sample, plus one divide when the cutoff or Q changes; see Resonant Filters for a comparison across the three.

For the cheaper (but range-limited and less accurate) Chamberlin state-variable filter, see Chamberlin Filter; for a 4-pole ladder, see Moog Ladder Filter.

Include

#include <q/fx/svf.hpp>

Declaration

   struct svf
   {
      static constexpr double default_q;     // 1/sqrt(2), maximally flat

                     svf(frequency f, float sps, double q = default_q);
                     svf(frequency f, float sps, duration tau);

      float          operator()(float x);    // process one sample; returns lowpass

      // Responses, valid after a call to operator()
      float          lowpass() const;
      float          bandpass() const;
      float          highpass() const;
      float          notch() const;
      float          peak() const;
      float          allpass() const;

      // Retune
      void           cutoff(frequency f, float sps);
      void           resonance(double q);
      void           normalized_resonance(float r);
      void           config(frequency f, float sps, double q);
      void           config(frequency f, float sps, duration tau);

      svf&           operator=(float y);
   };
   // Modal placement: the Q that a (f, tau) placement implies
   constexpr double q_from_decay(frequency f, duration tau);

Normalized resonance, a float in [0, 1].

A float.

Constructors and Assignment

Expression Semantics

svf(f, sps, q)

Construct with cutoff f, samples per second sps, and quality factor q.

svf(f, sps)

As above with q = svf::default_q (Butterworth).

svf(f, sps, tau)

Modal placement: a mode at f whose impulse ring decays by 1/e every tau (Q = pi * f * tau).

svf(b)

Copy construct from b.

a = b

Assign b to a.

a = y

Preset the state so the filter outputs the steady-state value y for a constant input y (clears the resonant/transient state).
C++ brace initialization may also be used.

Example

A resonant lowpass swept by an envelope, as in the Square Synth tutorial:

q::svf filter{1_kHz, sps, 2.0};               // cutoff, sps, Q

// ... in the processing loop, per sample:
filter.cutoff(q::frequency{cutoff_hz}, sps);  // sweep the cutoff
auto out = filter(in);                         // lowpass output
